================================================================
  PROJECT OVERVIEW
================================================================

This project implements a complete deep learning framework for 
maneuvering target tracking using attention mechanisms. It 
integrates seamlessly with the existing radar_simulation_v3.m 
dataset generator.

Key Features:
- LSTM/GRU-based temporal feature extraction
- Self-attention for global temporal dependencies
- Maneuver-aware attention for pattern recognition
- Dual-branch output: state estimation + maneuver classification
- Support for 2D/3D datasets
- Multi-receiver fusion capability

================================================================
  IMPLEMENTATION DETAILS
================================================================

1. Data Loader (data_loader_maneuver_tracking.m)
   - Loads scenarios from radar_simulation_v3.m dataset
   - Sliding window sample construction
   - Data normalization (minmax/zscore)
   - Automatic dataset splitting
   - Statistics computation

2. Network Architecture (maneuver_aware_tracking_network.m)
   - Input: [num_receivers × 3 × sequence_length]
   - LSTM/GRU layers (configurable depth)
   - Self-attention layer (multi-head)
   - Maneuver-aware attention
   - State estimation branch (9-dim output)
   - Maneuver classification branch (4 classes)

3. Attention Modules (attention_modules.m)
   - Self-attention implementation
   - Maneuver-aware attention
   - Positional encoding
   - Multi-head attention
   - Attention analysis tools

4. Training (train_maneuver_tracking.m)
   - Combined loss: α*MSE + β*CrossEntropy
   - Adam optimizer
   - Learning rate scheduling
   - Early stopping
   - Checkpoint saving

5. Evaluation (evaluate_tracking_network.m)
   - State estimation metrics (RMSE, MAE)
   - Classification metrics (accuracy, F1)
   - Trajectory visualization
   - Attention heatmaps
   - Error distribution analysis

6. Utilities
   - Coordinate transformations
   - Performance metrics
   - Visualization tools

================================================================
  CONFIGURATION PARAMETERS
================================================================

Data Configuration:
- sequence_length: 50 (input sequence length)
- stride: 10 (sliding window step)
- train_ratio: 0.7 (training set proportion)
- normalize_method: 'minmax' or 'zscore'

Network Configuration:
- hidden_size: 128 (LSTM hidden units)
- num_lstm_layers: 2 (LSTM depth)
- num_attention_heads: 8 (attention heads)
- dropout_rate: 0.2 (dropout probability)

Training Configuration:
- batch_size: 32
- num_epochs: 100
- learning_rate: 0.001
- state_loss_weight: 1.0
- maneuver_loss_weight: 0.5

================================================================
  TECHNICAL SPECIFICATIONS
================================================================

Requirements:
- MATLAB R2020a or later
- Deep Learning Toolbox
- Statistics and Machine Learning Toolbox
- (Optional) Parallel Computing Toolbox for GPU

Input Format:
- Samples: [N × 3 × 3 × 50]
  (N samples, 3 receivers, 3 features, 50 timesteps)
- State labels: [N × 9]
  (position + velocity + acceleration)
- Maneuver labels: [N × 1]
  (1=CV, 2=CA, 3=CT, 4=Singer)

Output Format:
- State prediction: [N × 9]
- Maneuver prediction: [N × 1]
- Attention weights: [N × seq_len × seq_len]

================================================================
  KNOWN LIMITATIONS
================================================================

1. MATLAB Version:
   - Some features require R2020a+
   - LayerGraph functionality may be limited

2. Training:
   - Custom training loop is simplified
   - Full backpropagation requires manual implementation
   - GPU support depends on available toolboxes

3. Dataset:
   - Requires pre-generated radar datasets
   - Fixed data format from radar_simulation_v3.m

4. Scalability:
   - Large networks may require significant memory
   - Batch size limited by available RAM
